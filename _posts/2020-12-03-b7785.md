회사에 있는 사람
<br>

:bulb: **풀이**<br>
- `hash`를 이용하여 풀었다.
- 받아온 사람의 이름이 hash에 존재하면 삭제하고, 존재하지 않으면 삽입한다.

<br>
<br>

> 코드
```python
import sys
input = sys.stdin.readline
n = int(input())
e = {}
for _ in range(n):
    name, action = input().split()
    if not name in e:
        e[name] = True
    else:
        del e[name]

print('\n'.join(sorted(e.keys(),reverse=True)))
```
